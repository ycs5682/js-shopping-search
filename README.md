# 要件定義

## 検索フォーム実装

[] 商品名の入力フォームがある  
[] キーワードがない場合は x ボタンを隠す  
[] キーワードを入力すると x ボタンが現れる  
[] 「Enter キー」を押すと検索結果が見える  
[] x ボタンをクリックするか検索を削除すると検索結果が消える

## 検索結果の実装

[] 検索結果が検索フォームの下にある  
[] 検索結果が見える  
[] x ボタンをクリックすると検索フォームが初期化され、検索結果が消える

## タブの実装

[]「おすすめキーワード」,「最近の検索」タブが検索フォームの下にある  
[] 初期設定で「おすすめキーワード」が選択されてある  
[] 各タブをクリックするとタブ下の内容が変わる

## 「おすすめキーワード」の実装

[] 番号,おすすめキーワードリストがタブの下にある  
[] リストでキーワードをクリックすると選択されたキーワードで検索結果画面に移動  
[] 検索フォームに選択されたおすすめキーワード設定

## 「最近の検索」の実装

[]「最近の検索」リストがタブの下にある  
[] リストでキーワードをクリックすると選択されたキーワードで検索結果画面に移動  
[] 検査の日付, x ボタンがある  
[] リストで x ボタンをクリックすると選択されたキーワードがリストから削除される  
[] 検索するたびに「最近の検索」に追加される
